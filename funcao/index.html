Funções

O que é uma função? Uma função é um bloco de código que realiza uma tarefa específica. Ela é usada para organizar, reaproveitar e evitar repetição de código. Pense nela como uma “máquina” que recebe algo, faz um trabalho e devolve um resultado.

Os parâmetros são como informações que a função precisa para funcionar. Eles são colocados entre parênteses () na definição da função.

Retorno de uma função (return) Às vezes, a função precisa devolver um resultado para ser usado em outro lugar. Usamos a palavra return.
